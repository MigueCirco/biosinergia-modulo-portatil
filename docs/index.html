Reglas del sistema

Lógica de control

  • Manual: control directo de relés desde la web.
  • Automático: el ESP32 decide según SETs.
  • CO2 alto activa ventilación.
  • Humedad baja activa humidificador.
  • Humidificador y ventilación no funcionan juntos si exclusión mutua está activada.
  • Todos los cambios quedan registrados en eventos e historial.
