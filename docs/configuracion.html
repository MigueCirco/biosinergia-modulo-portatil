Configuración de modo timer

El modo timer permite accionar humidificador y ventilación por ciclos de tiempo. Es útil como respaldo cuando un sensor falla o cuando se desea trabajar por horarios.

Configuración timer lista para editar.

Importante: el modo timer requiere firmware compatible. Si el firmware actual no interpreta mode: timer, la configuración quedará guardada en Firebase pero el ESP32 no la ejecutará todavía.

Humidificador y ventilación no funcionarán al mismo tiempo.

Reglas del sistema

Lógica de control

  • Manual: control directo de relés desde la web.
  • Automático: el ESP32 decide según SETs.
  • Timer: configuración web de ciclos y franjas horarias guardada en Firebase para firmware compatible.
  • CO2 alto activa ventilación.
  • Humedad baja activa humidificador.
  • Humidificador y ventilación no funcionan juntos si exclusión mutua está activada.
  • Todos los cambios quedan registrados en eventos e historial.
