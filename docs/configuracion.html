    Lo esencial para operar el cultivo día a día.

    💧 Humedad ambiente

    Rango recomendado para fructificación de Melena de León: alto, sin saturación constante.

    Si la humedad baja de este valor, el sistema intentará humidificar.

    Si la humedad supera este rango, dejará de humidificar.

    🌬️ CO2 / ventilación

    Si el sensor de CO2 falla, este control puede no ser confiable.

    Si el CO2 supera este rango, el sistema usará ventilación para renovar el aire.

    🌡️ Temperatura

    La temperatura crítica actúa como límite de seguridad.

    A partir de la temperatura crítica, el sistema prioriza bajar temperatura por seguridad.

    ⚙️ Reglas de funcionamiento

    Activar exclusión mutua evita que humidificador y ventilación se anulen entre sí.

    Si la exclusión mutua está activada, humidificador y ventilación no funcionarán al mismo tiempo.

    Lógica de control

    • Manual: control directo de relés desde la web.
    • Automático: el ESP32 decide según SETs.
    • Timer: configuración web de ciclos y franjas horarias guardada en Firebase para firmware compatible.
    • CO2 alto activa ventilación.
    • Humedad baja activa humidificador.
    • Humidificador y ventilación no funcionan juntos si exclusión mutua está activada.
    • Todos los cambios quedan registrados en eventos e historial.
